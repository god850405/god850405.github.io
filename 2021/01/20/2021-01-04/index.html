IIS架設網站

IIS SSL
部署
Publish Date:   2021-01-20

差不多在一個月前開始看影片( Vue.js 手牽手，一起嗑光全家桶 ) 自學 Vue.js 撰寫網頁，也同時學習使用 .Net 撰寫後端 API。不知道大家是否跟我一樣，辛辛苦苦寫好的一個網站，最期待的是能夠將自己的作品分享給大家，這篇文章將記錄下，如何使用微軟內建的IIS架設網站。

IIS

  • 安裝 IIS

  • 新增站台

部署 .Net 可能會遇到的問題

  • 檢查是否有安裝 .NET Core 主控套件組合

  • 如果程式碼確定無誤還是無法正常連線，可以嘗試檢查 web.config 有沒有設定好

開放外網連線

  • 必須擁有乙個固定 IP 中華電信 HiNet 固定 IP 申請教學
  • 設定路由器設定教學
  • 通訊埠轉發Port Forwarding
  • 防火牆增加輸入規則 ( 網站使用的埠號 ) 從 Windows 防火牆開啟 TCP Port 80

SSL

  • 申請SSL必須擁有自己的網域

  • 如需申請網域個人推薦Gandi，Gandi 網域教學

如果已經擁有自己的網域可以安裝免費SSL

  • 申請免費SSL

  • IIS - 安裝 SSL 憑證
